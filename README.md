# Blog Clone Project (Django)

A multi-user blog platform built with Django that allows users to create, manage, and interact with blog content, with admin-level moderation controls.

## Features

- User authentication (signup, login, logout)
- Create, edit, and delete blog posts
- Commenting system for user engagement
- Admin approval system for posts (content moderation)
- Role-based access (admin vs users)

## Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Architecture:** MVT (Model-View-Template)
- **Database:** SQLite / PostgreSQL

## Key Highlights

- Implemented content moderation with admin approval workflow  
- Designed structured MVT architecture for maintainability  
- Built interactive commenting system for user engagement  
- Managed user roles and permissions for secure operations
